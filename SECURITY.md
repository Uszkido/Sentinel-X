# Security Policy

## Supported Versions
Sentinel-X is in active development. Security updates are applied to the latest version.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | ✅ Yes            |

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability within Sentinel-X, please help us by reporting it responsibly.

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, please send an email to the project maintainers (Uszkido/Company) with:
1. Detailed description of the vulnerability.
2. Steps to reproduce the issue.
3. Potential impact.

We will acknowledge your report and work to provide a patch as quickly as possible.
