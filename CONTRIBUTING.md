# Contributing to Sentinel-X

Thank you for your interest in contributing to Sentinel-X! We are building the future of autonomous multimodal intelligence, and we welcome experts from AI, GIS, Defense, and Software Engineering.

## How Can I Contribute?

### 🤖 AI & Machine Learning
- **New Detection Skills**: Improve our YOLO/SAM models for specific object identification (e.g., specific vehicle types, tactical gear).
- **Vision Features**: Enhance our Super-Resolution or Re-ID algorithms.
- **NLP**: Integrate support for more African languages in our transcription and reasoning engines.

### 🚁 Drone & Robotics
- **MAVLink Integration**: Help us expand support for professional drone hardware (PX4, ArduPilot).
- **Swarm Logic**: Contribute new formation patterns or autonomous avoidance algorithms.

### 🌐 GIS & Frontend
- **Cesium Enhancements**: Improve our 3D Digital Twin rendering or add building footprint change detection.
- **Symbology**: Expand our MIL-STD-2525 implementation.

## Pull Request Process
1. Fork the repository.
2. Create a new feature branch.
3. Ensure your code follows the existing style and is well-documented.
4. Submit your PR with a detailed description of the changes.

## Reporting Issues
Please use the GitHub Issue tracker to report bugs or suggest new "Expert" features.
